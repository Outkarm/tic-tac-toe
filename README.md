## ğŸ’» Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project, you need to have the following installed:

- Ruby (version 3.2.1)
- Node.js (version 18.16.0)
- Yarn (optional but recommended)
- PostgreSQL

### Setup

Clone this repository to your desired folder:

```sh
git clone git@github.com:Outkarm/tic-tac-toe.git
```

### Install

Install the required Node.js packages:

```sh
npm install
```

### Usage

Start the server:

```sh
npm run dev
```

Open your browser and visit `http://localhost:5173` to see the application running.

### Deployment

This project can be deployed using your preferred hosting platform or service. Follow the deployment instructions provided by the platform of your choice.
